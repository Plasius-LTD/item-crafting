# Item Crafting Package Bootstrap

## Goal

Provide a package-standard bootstrap for item-crafting authority.

## Initial Surface

- package descriptor and feature-flag metadata
- item-crafting access-state contract
- portable handoff-host contract
- bounded retry-policy and handoff contracts
- access-state helper
- demo and test scaffolding

## Exclusions

- Player System orchestration
- crafting UI
- renderer behavior
